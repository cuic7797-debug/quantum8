# Quantum8 使用说明

## 快速开始

### 1. 首次使用
1. 打开网站 https://5811938f.quantum8.pages.dev/
2. 点击首页的「同步数据」按钮获取最新开奖数据
3. 等待数据加载完成（约5-10秒）

### 2. 核心功能导航

#### 查看数据
- **首页**：最新开奖 + 热号/冷号概览
- **走势分析**：15种专业图表，点击展开查看详情
- **历史开奖**：搜索/筛选历史期号

#### 智能选号
1. 选择玩法（选一到选十）
2. 选择投注方式（单式/复式/胆拖）
3. 选择策略（保守/均衡/激进）
4. 点击「生成推荐号码」
5. 可选：保存/复制/分享结果

#### AI 策略生成器
1. 选择策略模板（6种预设）
2. 调整参数（可选）
3. 点击「生成推荐号码」
4. 查看AI算法评分面板

#### 策略回测
1. 输入或选择号码组合
2. 设置回测参数
3. 点击「开始回测」
4. 查看ROI/命中率/风险指标

### 3. 快捷键

按 `?` 键随时查看所有快捷键：
- `1`-`6`：快速跳转常用页面
- `b`：AI策略生成器
- `d`：数据管理
- `t`：时序分析
- `p`：号码画像

### 4. 数据同步

数据每天自动从福彩官网更新。如果数据不是最新的：
1. 点击首页的「刷新数据」按钮
2. 或在数据管理页面点击「同步数据」

### 5. 主题切换

点击导航栏顶部的主题切换按钮，可在深色/浅色模式间切换。

### 6. 账户功能

注册登录后可以：
- 云端保存选号记录
- 云端保存自定义策略
- 跨设备同步数据

## 算法说明

本平台使用以下算法进行分析：

| 算法 | 用途 | 说明 |
|---|---|---|
| 马尔可夫链 | 号码出现概率 | 分析状态转移规律 |
| 贝叶斯推断 | 后验概率 | 结合先验和观测数据 |
| 信息熵 | 不确定性评估 | 评估号码可预测性 |
| 集成评分 | 综合评分 | 多算法加权融合 |
| 趋势回归 | 趋势预测 | 线性回归分析 |
| 遗传算法 | 号码生成 | 进化优化选号 |
| 聚类分析 | 号码分群 | 热/温/凉/冷分类 |
| 布林带 | 异常识别 | 识别异常区间 |
